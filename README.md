## 🤖 What It Does

Paste any text and instantly get:
- **Sentiment** — Positive / Negative / Neutral
- **Emotion** — Joy, Anger, Sadness, Fear, Surprise, Calm
- **Polarity Score** — from -1.0 (very negative) to +1.0 (very positive)
- **Subjectivity Score** — how opinion-based the text is
- **Word & Sentence count**

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/PrinceKChaudhari/AI-Sentiment-Analyzer.git
cd AI-Sentiment-Analyzer

# 2. Install dependencies
pip install -r requirements.txt
python -m textblob.download_corpora

# 3. Run the app
python app.py

# 4. Open browser
# http://localhost:5000
```
